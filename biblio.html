Ce site web est réalisé en HTML 5 et CSS 3 à l'aide du logiciel gratuit Notepad ++

- Les illustrations sont des créations originales, réalisées à l'aide de pixlr.com et du logiciel gratuit Paint.NET

- Les images vectorielles viennent de Canvas éducation pour l'enseignement - export en .svg

- Les codes CSS et HTML sont inspirés puis adaptés à partir de nombreuses contributions publiées sur codePEN

- Les images 3D et modélisations sont réalisées avec SketchUp make 2017

- L'hébergement du site est réalisé sur GitHUB

Un grand merci à mes professeurs sans qui le projet n'aurait pas abouti.
